# Smart Campus Insights

Streamlit app for campus data analytics.

## Quick Start

1. `pip install -r requirements.txt`
2. `streamlit run smart_campus_insights_app.py`

## Data Files
- attendance_logs.xlsx
- event_participation.xlsx
- lms_usage.xlsx

## Features
- Attendance visualizations
- Event stats
- LMS metrics
- ML engagement prediction

## Deploy
Fork repo, connect Streamlit Cloud.
